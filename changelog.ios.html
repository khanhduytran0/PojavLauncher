System Support in future releases

crystall1nedev: We've made some amazing advancements in the past few months with the launcher, however this announcement is for something else. I revisited an old conversation with Duy about iOS version support in PojavLauncher iOS. After digging into the facts, such as feature parity, the userbase, and the user experience, we have come to the following conclusion:

In a future release, PojavLauncher will require iOS 14.0 or later.
You can learn more about how this will happen over time with the GitHub issue linked here

PojavLauncher iOS: Non-jailbroken devices, Java 17, "Shulk Catalyst", and more

crystall1nedev: We've gotten pretty far with unjailbroken PojavLauncher in the past few days thanks to DuyKhanhTran's brilliance, and we are planning for 2.1 of the launcher to include an unjailbroken package! However, I just want to say a few things:...

Sideloading the ipa when jailbroken has all of the features you know--as well as a new directory system from the native Files app! Because of the feature parity when jailbroken, there's no need to include a special jailbroken package, and we will not be providing one with PojavLauncher 2.2 or later.
â€¢ On jailbroken devices, AppSync Unified can be used to install the .ipa without an Apple ID.
